# guitar.io

## Development

To test the application locally, you can run a simple web server from the project's root directory. This is recommended over opening the `index.html` file directly in your browser, as some features may not work as expected due to browser security policies.

### Running a Local Server

1.  Open a terminal or command prompt in the root directory of the project.
2.  If you have Python installed, you can use its built-in HTTP server. Run the following command:

    ```bash
    python -m http.server
    ```

    If you have Python 2, the command is:

    ```bash
    python -m SimpleHTTPServer
    ```

3.  Open your web browser and navigate to `http://localhost:8000`.

The application will now be running on your local machine.
